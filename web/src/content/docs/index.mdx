## Manual de nu

Esta es la documentación de uso de `nu`: qué es, cómo se instala y **una
referencia función a función de la API del core** con ejemplos ejecutables.

<CardGrid stagger>
	<Card title="Empieza en un minuto" icon="rocket">
		Instala un único binario estático —sin Node, sin toolchain— y ejecuta
		tu primer script Lua con `nu -e`. Ver [Instalación](/nu/empezando/instalacion/).
	</Card>
	<Card title="Todo es una extensión" icon="puzzle">
		El core no sabe lo que es un agente. El loop del agente, el chat y los
		providers son extensiones Lua sobre la API pública. Ver
		[Conceptos clave](/nu/empezando/conceptos/).
	</Card>
	<Card title="Una API pequeña y sagrada" icon="approve-check">
		Trece namespaces bajo `nu`, async por corrutinas y errores
		estructurados. Crece solo por adición. Ver
		[Convenciones](/nu/referencia/convenciones/).
	</Card>
	<Card title="Lua decide, Go ejecuta" icon="setting">
		Búsqueda a escala de repo, diff, markdown, highlighting y streaming HTTP
		son primitivas Go paralelas por dentro. Ver
		[nu.search](/nu/referencia/search/) y [nu.text](/nu/referencia/text/).
	</Card>
</CardGrid>

:::note[Sobre esta documentación]
La fuente de verdad de la API es [`docs/api.md`](https://github.com/dbareagimeno/nu/blob/main/docs/api.md)
en el repositorio (la "superficie sagrada" v1). Este manual la presenta de
forma orientada a tareas, con ejemplos. Si algo discrepa, manda `docs/api.md`.
:::
